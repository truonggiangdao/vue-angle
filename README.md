# Vue Angle Picker

### Introduction

This is a small project to create the angle picker as I want because I couldn't find it online.


### Instruction

Install:

```
npm install vue-angle
```

Use:

```
import Vue from 'vue';
import AnglePicker from 'vue-angle';
vue.use(AnglePicker);
```


### Browser Support

Tested on Chrome and Safari
